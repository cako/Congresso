Os dados acima foram todos obtidos através da API pública RESTful que pode ser encontrada aqui. Contudo, para não sobrecarregar os servidores e não estar sujeito a velocidade de redes, é conveniente salvar os dados. Isto é feito usando a flag "write", e especificando um local para guardar os arquivos. O padrão é o diretório local. Não cobriremos esta parte, mas vamos supor que temos os dados já previamente baixados. Para que estes sejam usados, devemos definir o local onde os arquivos são guardados.

Para finalizar os exemplos, vamos ver a distribuição dos estados em cada legislatura. Abaixo vamos criar uma uma lista que cada elemento é um dicionário. Esse dicionário contem pares UF - no. de senadores.

In [67]: 
leg_UFs = []
for leg in legs:
    UFs = {}
    for sen in leg.senadores:
        for mand in sen.mandatos:
            if mand.legini == leg.leg:
                UFs[mand.UF] = UFs.get(mand.UF, 0) + 1
                break
    leg_UFs.append(UFs)
